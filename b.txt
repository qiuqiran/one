git指南

创建新仓库 git init

检出仓库 git clone git@github.com:qiuqiran/one.git

检查 git status

添加 git add a.txt

提交 git commit -m "第一次修改"

推送 git push origin master

连接远程服务器 git remote add origin <server>

创建two分支并切换过去 git checkout -b two

查看有多少分支 git branch

删除分支 git branch -d two

更新 git pull
合并 git merge two
